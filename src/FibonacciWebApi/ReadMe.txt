Este programa consiste en una API que devuelve una respuesta HTTP en caso de recibir valores no contemplados por el algoritmo de Fibonacci (no numeros, numeros negativos o fracciones).
En caso de recibir un dato erroneo, se devolvera el codigo de HTTP acorde.
En caso de correcto procesado de los datos introducidos, se devolvera el resultado numerico.


Ruta para ejecucion https://localhost:#####/api/Fibonacci/##

NOTA: Se reemplazan los simbolos # de manera que el primer campo corresponde con el localhost de la PC del usuario y el segundo corresponde al id o numero n que se ingresa para el calculo de fibonacci
